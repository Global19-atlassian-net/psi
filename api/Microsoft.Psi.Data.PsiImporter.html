    Remarks

    Reads either at the full speed allowed by available resources or at the desired rate specified by the Pipeline. Instances of this class can be created using the Open(Pipeline, String, String) method. The store metadata is available immediately after open (before the pipeline is running) via the AvailableStreams property.
