    IsAutoGeneratedNullableMember

    Gets or sets a value indicating whether this stream is an auto-generated nullable member.

    Declaration
    [Browsable(false)]
public bool IsAutoGeneratedNullableMember { get; protected set; }
    Property Value
    Type Description
    Boolean
    Remarks

    When stream members are generated for value types, if any of the ancestor streams are reference type, we generate a corresponding nullable value type for the stream member (instead of the actual value type). This is done so that when the ancestor object is null, we can still produce values (in this case null as well) for the member.
